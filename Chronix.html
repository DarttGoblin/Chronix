For Brave users, Brave disables Chrome extension APIs like chrome.storage by default for privacy reasons, leading to unexpected behaviours and errors in Chronix, to fix this issue for Brave browser:

1. Go to Brave Settings.
2. Navigate to extensions section in the sidenav bar.
3. Enable "Allow Google Login for Extensions".
3. Restart Brave and Reload the Extension.

This allows Chrome's extension APIs to work properly, and Chronix will run smoothly. Enjoy!
